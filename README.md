## ✦ 說明 ✦

遊戲使用 Unity 遊戲引擎製作，透過 GitHub 將遊戲上傳至網頁，遊戲**僅支持電腦操作**，遊玩時進入全螢幕，按下任意鍵後即可開始，使用鍵盤 D F J K 鍵控制，當音符落下瞬間按下對應按鍵即可得分，長音符則須常按至音符末端，音樂結束即遊戲結束。
